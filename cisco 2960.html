Mise à jour du firmware sur les switches Cisco 2960

Les switches Cisco 2960 sont couramment utilisés dans les petites et moyennes entreprises (PME) pour leur fiabilité et leur coût abordable. Garder le firmware à jour sur ces switches est important pour maintenir la sécurité et les performances du réseau.

Prérequis :

  • Switch Cisco 2960
  • Serveur TFTP
  • Câble Ethernet
  • Ordinateur avec logiciel d'émulation de terminal

Étapes :

1. Télécharger le firmware :

  • Va sur le site web de Cisco et navigue vers la page de téléchargement pour ton modèle spécifique de switch Cisco 2960.
  • Sélectionne la dernière version de firmware disponible (General Availability - GA).
  • Télécharge l'image du firmware sur un serveur TFTP.

2. Se connecter au switch :

  • Connecte une extrémité d'un câble Ethernet à un port Ethernet du switch et l'autre extrémité à un port Ethernet de ton ordinateur.
  • Ouvre le logiciel d'émulation de terminal sur ton ordinateur.
  • Configure le logiciel d'émulation de terminal avec les paramètres suivants :
    • Speed : 100 Mbps
    • Duplex : Full
    • Flow control : None
  • Entre les commandes suivantes dans le terminal :
    • Username : admin
    • Password :
    Remplace par le mot de passe de l'utilisateur admin sur le switch.

3. Copier le firmware sur le switch :

  • Entre la commande suivante dans le terminal : 
    copy tftp:/// flash:
    Remplace par l'adresse IP du serveur TFTP et par le nom du fichier de l'image du firmware.

4. Vérifier le firmware :

  • Entre la commande suivante dans le terminal : 
    show flash:
    Vérifie que l'image du firmware est listée dans la sortie.

5. Redémarrer le switch :

  • Entre la commande suivante dans le terminal : 
    reload
    Le switch va redémarrer et le nouveau firmware sera chargé.

6. Vérifier le nouveau firmware :

  • Entre la commande suivante dans le terminal : 
    show version
    La sortie de cette commande montrera la nouvelle version du firmware.

Notes supplémentaires :

  • Il est recommandé de sauvegarder la configuration du switch avant de mettre à jour le firmware.
  • Ne mets pas à jour le firmware pendant les heures de pointe d'utilisation du réseau.
  • Si tu rencontres des problèmes pendant le processus de mise à jour, consulte la documentation Cisco ou contacte le support Cisco.

Conseils :

  • Utilise un onduleur (UPS) pour alimenter le switch et le serveur TFTP en cas de coupure de courant.
  • Utilise un câble Ethernet dédié pour la connexion TFTP afin d'éviter la congestion du réseau.
  • TFTP est un protocole simple, mais il existe plusieurs méthodes alternatives pour télécharger le firmware sur les switches Cisco, comme HTTP, HTTPS et USB.

En suivant ces étapes, tu peux mettre à jour le firmware de tes switches Cisco 2960 de manière sûre et facile.
